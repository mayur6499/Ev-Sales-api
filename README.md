# Electric Vehicle Sales Analysis & Forecasting
## Project Overview:
This project analyzes Electric Vehicle (EV) sales data and builds a machine learning model to predict future EV sales.

## Business Problem
* Forecast EV demand
* Understand market growth

## Dataset:
### columns:15
'record_id',
            'country',
            'region',
            'ev_brand',
            'vehicle_type',
            'battery_capacity_kwh',
            'vehicle_range_km',
            'charging_time_hours',
            'charging_stations',
            'govt_incentives',
            'avg_ev_price_usd',
            'year',
            'fuel_price_index',
            'co2_regulation,
            'gdp_growth'

## Project Workflow:
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Trend & Growth Analysis
4. Machine Learning / Forecasting Model
5. Model Evaluation
6. Business Insights

##  Machine Learning Model:
* Algorithm: Linear Regression / Random Forest
* Input Features: Year, Market Share, Growth Rate
* Target: EV Sales

## Key Insights:
1. EV sales are growing year by year
2. Government policies and awareness drive adoption
3. Future sales show strong upward trend
